# 🌍 AI Trip Planner

An intelligent AI-powered travel planning application that creates personalized travel itineraries based on your destination, budget, travel dates, interests, and preferences. Built to simplify trip planning by generating complete travel schedules within seconds.

---

## ✨ Features

* 🤖 AI-generated personalized travel itineraries
* 📍 Destination recommendations
* 🏨 Hotel suggestions based on budget
* 🍽️ Restaurant recommendations
* 🗺️ Daily travel schedule
* 💰 Budget estimation
* 🌦️ Weather information
* 📸 Must-visit attractions
* 🚗 Transportation suggestions
* ❤️ Save and manage trips
* 📱 Fully responsive design

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/aman36yt/ai-trip-planner.git
```

### Navigate to the project

```bash
cd ai-trip-planner
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env` file and add:

```env
GEMINI_API_KEY=your_api_key
GOOGLE_MAPS_API_KEY=your_google_maps_key
MONGODB_URI=your_database_url
JWT_SECRET=your_secret_key
```

### Start the development server

```bash
npm run dev
```

The application will run on:

```
http://localhost:3000
```

## 🧠 How It Works

1. Enter your destination.
2. Select travel dates.
3. Set your budget.
4. Choose travel preferences.
5. AI analyzes your inputs.
6. Generates a complete travel itinerary.
7. View hotels, attractions, restaurants, and daily plans.
8. Save or share your trip.
